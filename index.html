Help

Open the bottom bar by clicking the plus(+) button.
Drag components from the bar onto the canvas.
Delete them by dragging them out of the canvas.
Drag from inputs or outputs to create a wire.
Delete wires by clicking on the wire.

COMPUTOR:
Use your mouse to pan and zoom on the canvas.
Right-click a component to open its edit menu.
Ctrl/Cmd+Z to undo.
Ctrl/Cmd+Y to redo.

MOBILE:
Drag to pan and pinch to zoom.
Long-press a component to open its edit menu.

Click the button below to check out some projects.

Sources

Voltage Source

Provides a constant voltage between two points in your circuit.

Current Source

Provides a constant current flow regardless of load.

AC Voltage Source

Works like a voltage source, but alternates flow at a set interval in a sine wave.

AC Current Source

Works like a current source, but alternates flow at a set interval in a sine wave.

Ground

Ground acts as a simplified, movable end for your circuit.

Signal Control

Switch

Opens or closes a connection when clicked to control current flow.

Button

Temporarily connects a circuit while being pressed, then releases.

Clock Generator

Sends repeating on/off signals with a set interval for both on and off.

Pulse Generator

Works like a Clock Generator but can fire a number of pulses.

Transistor

Allows current from one or both inputs to flow if the top input has enough voltage.

Diode

Allows current to flow in one direction only.

Passive Control

Resistor

Limits the amount of current flowing through a circuit.

Fuse

Breaks the circuit if too much current flows, protecting components.

Capacitor

Stores and releases electrical energy when needed.

Inductor

Stores energy in a magnetic field when current passes through.

Output

LED

Emits light when current flows through it; it is efficient and bright.

Light Bulb

Glows by heating a filament; it turns on and off more gradually.

Buzzer

Makes a sound when powered; its useful for alerts.

Segment Display

Shows numbers using 7 segments and 1 decimal point input.

Gates

Merger

Combines multiple currents into 1 path.

Splitter

Splits a current into multiple paths.

And

Flows only if all inputs are on.

Or

Flows if at least one input is on.

Nand

Opposite of AND. Flows if at least 1 input is off. Needs a set output power.

Nor

Opposite of OR. Flows only if all inputs are off. Needs a set output power.

Xor

Flows if an odd number of inputs are on.

Xnor

Opposite of XOR. Flows if an even number of inputs are on. Needs a set output power.

Not

Outputs the opposite of the input signal. Needs a set output power.

Advanced

D Flip-Flop

Only sets the output to the first input if the top input(clock) is pulsed on.

T Flip-Flop

Toggles if the first input is on and the top input(clock) is pulsed on. Needs a set output power.

SR Flip-Flop

Works like an SR Latch but only changes when the top input(clock) is pulsed on. Needs a set output power.

JK Flip-Flop

Works like an SR Flip-Flop but allows both inputs on at the same time, which makes the output toggle. Needs a set output power.

SR Latch

The first input sets output to on, the second one resets output to off. Don't input both signals. Needs a set output power.

Binary Counter

Has an input(clock) that increments a binary output. Needs a set output power.

Mux

Sends one of many inputs to the output based on the top select signal.

Demux

Sends the input to one of many outputs based on the top select signal.

Encoder

Turns an input into a binary number output.

Decoder

Turns binary input into an output. Needs a set output power. Can be used on the Binary Counter.

Display Decoder

Turns binary input into a number display output with the fifth input being for the decimal. Needs a set output power. Can be used for the Segment Display.

Annotation

Text

Adds labels or notes to explain your circuit. The point can be used to move it.

Arrow

Draws arrows to point out parts of your circuit. The starting point and the arrowhead can both be moved.
